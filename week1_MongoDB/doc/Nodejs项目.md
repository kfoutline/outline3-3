# Nodejs项目

## 后台管理系统
>在二阶段项目的基础上实现一个电商网的后台管理系统

### 技术要求
* 后端语言：Nodejs+Express/Koa
* 数据库：MongoDB/mySQL
* UI框架：使用但不限于bootstrap、layui等框架

### 功能要求
* 数据可视化
    * 数据统计
* 商品管理
    * 列表展示
    * 添加
    * 编辑
    * 删除
* 用户管理
    * 添加
    * 编辑
    * 删除
* 订单管理
    * 订单展示
    * 订单状态修改
        * 未付款
        * 付款成功
        * 发货
        * 签收
        * 完成订单
