#NPM(Node Package Manager包管理)
Node.js 的包管理器 npm，是全球最大的开源库生态系统，随着nodejs一起安装（`npm -v`查看安装版本）

##安装包npm install
* 普通安装 `npm install <package>`
* 安装指定版本 `npm install <package>@xx.xx.x`
* 安装到依赖
    * `npm install <package> --save`
    * `npm install <package> --save-dev`
